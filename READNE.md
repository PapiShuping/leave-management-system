# Leave Management System

> A leave management system with ASP.NET backend and flutter mobile app

## Project Status
- Building project files

##Tech stack
**Backend:**
-ASP.NET Core 8.0 Web API
-SQL Server LocalDB
-Entity Framework Core
-JWT Authentication

**Mobile:**
-Flutter 3.35.6
-Provider State management
-Rest API Intergration

## Planned Features

### Employee Features(Mobile app)
-Login with email and password
-Apply for leave with documentupload
-View Leave history
-Check leave balance
-Receive Notification

### HR/Manager Features (Web Portal)
-Review leave requests
-Approve/Reject with comments
-Manage employees
-Generate reports
-Manage holidays
-Dashboard

### Leave Types
-Annual leave(21 days)
-Sick leave(30 days)
-Family Responsibility(3 days)
-Maternity leave(130 days)

## Getting Started

### Prerequisities
- .NET 8.0 SDK
- SQL Server LocalDB
- Flutter 3.xx
- Visual Studio 2022
